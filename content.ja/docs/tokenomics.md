# トークノミクス

イーサウェイ のトークンモデルは、予測可能で退屈になるように設計されています。複雑なベスティングの崖も、
ガバナンスが制御するインフレも、予想外の出来事もありません。

## トークン: ETE

| 項目 | 値 |
|---|---|
| ティッカー | **ETE** |
| 最小単位 | `WEI` |
| 小数桁数 | 6（1 ETE = 1,000,000 WEI） |
| ジェネシス供給量 | 7,000,000,000 ETE（70億） |
| 最大供給量 | 10,000,000,000 ETE（100億） |
| 年間発行量 | 100,000,000 ETE / 年（1億） |
| 発行期間 | 30年 |
| ブロックあたり発行量 | 約 15.84 ETE / ブロック |
| コミュニティ税 | **0%** |

## 供給スケジュール

このチェーンはジェネシスから、毎年 **100,000,000 ETE** を固定で発行し、それを 30年間続けます。
これはインフレ率ベースではなく、総供給量とは無関係の固定された絶対量です。

```
0年目    7,000,000,000 ETE（ジェネシス）
1年目    7,100,000,000 ETE
5年目    7,500,000,000 ETE
10年目   8,000,000,000 ETE
20年目   9,000,000,000 ETE
30年目  10,000,000,000 ETE（最大供給量に到達）
```

30年後、発行は停止します。その後、チェーンは取引手数料のみで運営を続けます。

## ブロック報酬

約 6.3 秒のブロック時間の場合（CometBFT のデフォルト値）:

```
年間ブロック数 ≈ 5,000,000
ブロックあたり発行量 ≈ 100,000,000 / 5,000,000 = 20 ETE
```

> [!NOTE]
> 仕様では約 15.84 ETE/ブロックとされており、これは年間約 6,313,131 ブロック
> （約 5 秒の平均ブロック時間に近い）に相当します。実際のブロック報酬は、
> ライブチェーンで観測されるブロック時間によって変わります。

## 分配

すべてのブロック報酬は、ステーキング比率に応じて**バリデータとその委任者**に還元されます。
コミュニティプール税、財団の取り分、プロトコル手数料はありません。

```
ブロック報酬
  └─ 100% バリデータ集合へ
       ├─ バリデータ手数料（バリデータごとに個別設定）
       └─ 委任者報酬（ステーキング比率に応じて）
```

## 単位に関する注記

オンチェーンでは、すべての金額は `WEI`（最小単位）で表されます。`1000000 WEI` の取引を見た場合、それは
`1 ETE` です。ウォレットとエクスプローラーは、読みやすさのために `ETE` 単位で表示します。
