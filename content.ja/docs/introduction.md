# はじめに

> **退屈でシンプルなブロックチェーンこそ、良いブロックチェーンです。**

Ethwei は Cosmos SDK ベースのプルーフ・オブ・ステーク（PoS）チェーンです。その役割はただ一つ。正しく
動作し、動作し続け、邪魔をしないことです。

## Ethwei とは

- Cosmos-SDK / CometBFT チェーン
- 固定された予測可能なトークン発行 — 予想外の出来事がない
- コミュニティ税 0% — バリデータと委任者が稼いだものをそのまま受け取る
- オープンなバリデータ集合 — 誰でもノードを運用できる

## Ethwei でないもの

- DeFi プラットフォーム
- スマートコントラクトチェーン（現時点では）
- ガバナンス偏重のプロトコル
- トークノミクスを気まぐれに変える存在

## 現在のステータス

Ethwei は現在**テストネット**（`ethwei-testnet-1`）段階です。メインネットは、テストネットが十分に検証
された後に開始される予定です。

テストネットのトークンに金銭的価値はありません。
