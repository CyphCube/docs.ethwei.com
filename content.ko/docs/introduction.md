# 소개

> **지루하고 단순한 블록체인이 좋은 블록체인입니다.**

이더웨이(Ethwei)는 Cosmos SDK 기반의 지분증명(PoS) 체인입니다. 그 역할은 단 하나입니다: 올바르게 작동하고,
계속 작동하며, 방해가 되지 않는 것.

## 이더웨이란

- Cosmos-SDK / CometBFT 체인
- 고정되고 예측 가능한 토큰 발행 — 예상치 못한 일이 없음
- 0% 커뮤니티 세금 — 검증인과 위임자가 번 것을 그대로 가져감
- 개방형 검증인 집합 — 누구나 노드를 운영할 수 있음

## 이더웨이가 아닌 것

- DeFi 플랫폼
- 스마트 컨트랙트 체인 (현재로서는)
- 거버넌스 중심의 프로토콜
- 토큰 이코노미를 마음대로 바꾸는 프로젝트

## 현재 상태

이더웨이는 현재 **테스트넷**(`ethwei-testnet-1`) 단계입니다. 메인넷은 테스트넷이 충분히 검증된 후에
시작될 예정입니다.

테스트넷 토큰은 어떠한 금전적 가치도 없습니다.
