# 토큰 이코노미

이더웨이의 토큰 모델은 예측 가능하고 지루하도록 설계되었습니다. 복잡한 베스팅 절벽도, 거버넌스가 통제하는
인플레이션도, 예상치 못한 일도 없습니다.

## 토큰: ETE

| 속성 | 값 |
|---|---|
| 티커 | **ETE** |
| 최소 단위 | `WEI` |
| 소수 자릿수 | 6 (1 ETE = 1,000,000 WEI) |
| 제네시스 공급량 | 7,000,000,000 ETE (70억) |
| 최대 공급량 | 10,000,000,000 ETE (100억) |
| 연간 발행량 | 100,000,000 ETE / 년 (1억) |
| 발행 기간 | 30년 |
| 블록당 발행량 | 약 15.84 ETE / 블록 |
| 커뮤니티 세금 | **0%** |

## 공급 일정

이 체인은 제네시스부터 매년 **100,000,000 ETE**를 고정적으로 발행하며, 이를 30년간 지속합니다.
이는 인플레이션율 기반이 아니라 총공급량과 무관한 고정된 절대 수량입니다.

```
0년차    7,000,000,000 ETE (제네시스)
1년차    7,100,000,000 ETE
5년차    7,500,000,000 ETE
10년차   8,000,000,000 ETE
20년차   9,000,000,000 ETE
30년차  10,000,000,000 ETE (최대 공급량 도달)
```

30년 후에는 발행이 중단됩니다. 이후 체인은 거래 수수료만으로 계속 운영됩니다.

## 블록 보상

약 6.3초의 블록 시간에서(CometBFT 기본값):

```
연간 블록 수 ≈ 5,000,000
블록당 발행량 ≈ 100,000,000 / 5,000,000 = 20 ETE
```

> [!NOTE]
> 사양에는 약 15.84 ETE/블록으로 되어 있으며, 이는 연간 약 6,313,131 블록
> (약 5초의 평균 블록 시간에 더 가까움)에 해당합니다. 실제 블록 보상은
> 라이브 체인에서 관측된 블록 시간에 따라 달라집니다.

## 분배

모든 블록 보상은 스테이킹 비율에 따라 **검증인과 그 위임자**에게 돌아갑니다.
커뮤니티 풀 세금, 재단 몫, 프로토콜 수수료가 없습니다.

```
블록 보상
  └─ 100% 검증인 집합에게
       ├─ 검증인 수수료 (검증인별로 개별 설정)
       └─ 위임자 보상 (스테이킹 비율에 따라)
```

## 단위에 관한 참고

온체인에서는 모든 금액이 `WEI`(최소 단위)로 표시됩니다. `1000000 WEI` 거래를 보게 된다면 그것은
`1 ETE`입니다. 지갑과 익스플로러는 가독성을 위해 `ETE` 단위로 표시합니다.
