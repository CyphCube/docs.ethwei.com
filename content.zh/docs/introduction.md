# 简介

> **一条无聊而简单的区块链，就是一条好区块链。**

以太维（Ethwei）是一条基于 Cosmos SDK 的权益证明（PoS）区块链。它只有一个任务：正确运行、持续运行，并且不碍事。

## 以太维是什么

- 一条 Cosmos-SDK / CometBFT 区块链
- 固定、可预测的代币增发——没有意外
- 0% 社区税——验证人和委托人保留他们的全部收益
- 开放的验证人集合——任何人都可以运行节点

## 以太维不是什么

- 一个 DeFi 平台
- 一条智能合约链（目前而言）
- 一个治理繁重的协议
- 一个会随意改变其代币经济的项目

## 当前状态

以太维目前处于**测试网**阶段（`ethwei-testnet-1`）。主网将在测试网经过充分实战检验后启动。

测试网代币没有任何货币价值。
