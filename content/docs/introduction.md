# Introduction

> **A boring and simple blockchain is a good blockchain.**

Ethwei is a Cosmos SDK proof-of-stake chain. It has one job: run correctly, keep running, and get
out of the way.

## What Ethwei is

- A Cosmos-SDK / CometBFT chain
- Fixed, predictable token emission — no surprises
- 0% community tax — validators and delegators keep what they earn
- Open validator set — anyone can run a node

## What Ethwei is not

- A DeFi platform
- A smart-contract chain (for now)
- A governance-heavy protocol
- Something that will pivot its tokenomics

## Current status

Ethwei is currently in **testnet** (`ethwei-testnet-1`). Mainnet is planned once the testnet
is sufficiently battle-tested.

Testnet tokens have no monetary value.
